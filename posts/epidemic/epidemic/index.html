SIR model

The SIR model is a couple of differential equations that work together. That system is

\begin{equation} \Large\begin{array}{c} \frac{dS}{dt}=-\beta SI \\\
\frac{dI}{dt}=\beta SI-vI \\\
\frac{dR}{dt}=vI \end{array} \end{equation}

In this equation, $S$ are the susceptible individuals, $I$ are the infected individuals and $R$ are the Recovered individuals. $\beta$ are the rate of individuals per period that become infected and $v$ are the rate of individuals per period that become recovered.

We can look that $\beta$ is multiplied by $S$ and $I$ because this rate depends of both individuals, while $v$ only is multiplied by $I$. In this model, the total poblation is conserved (there are no births or deaths).

I’m gonna use the following libraries

import matplotlib.pyplot as plt
import numpy as np
from numba import njit
from jupyterthemes import jtplot
jtplot.style(theme='monokai')

You can discretized the equations with Euler method. That say that:

\begin{equation} \frac{dx}{dt}\approx\frac{x_{i+1}-x_i}{\Delta t} \end{equation}

In this case, the system of equations are:

\begin{equation} \Large\begin{array}{c} S_{i+1}=S_i+\Delta t\left(\frac{dS}{dt}\right) \\\
I_{i+1}=I_i+\Delta t\left(\frac{dI}{dt}\right) \\\
R_{i+1}=R_i+\Delta t\left(\frac{dR}{dt}\right) \end{array} \end{equation}

Using the system of equations from the SIR model, we had

\begin{equation} \Large\begin{array}{c} S_{i+1}=S_i-\Delta t\beta S_iI_i \\\
I_{i+1}=I_i+\Delta t(\beta S_iI_i-vI_i) \\\
R_{i+1}=R_i+\Delta tvI_i \end{array} \end{equation}

We can rewrite the system of equations as:

\begin{equation} \Large\begin{array}{c} S_{i+1}=S_i(1-\Delta t\beta I_i) \\\
I_{i+1}=I_i(1+\Delta t(\beta S_i-v)) \\\
R_{i+1}=R_i+\Delta tvI_i \end{array} \end{equation}

The followind Python function solve the past system

@njit() #For solve the SIR model
def sir(N,dt,beta,v,S0,I0,R0):
    S,I,R=IC(N,S0,I0,R0)
    for i in range(0,N-1):
        S[i+1]=S[i]*(1-dt*beta*I[i])
        I[i+1]=I[i]*(1+dt*(beta*S[i]-v))
        R[i+1]=R[i]+dt*v*I[i]
    return S,I,R

@njit()
def IC(N,S0,I0,R0):
    S=np.zeros(N)
    I=np.zeros(N)
    R=np.zeros(N)
    #Put the initial conditon for every system
    S[0]=S0
    I[0]=I0
    R[0]=R0
    return S,I,R

We can observe how it works the system. We know that our system is close, that means $N=S+I+R$ for every $t$.

To observe how it works, we can suggest the next values

  • $N=10,000$
  • $t_f=100$ days
  • $I_0=10$
  • $R_0=5$
  • $S_0=N-I_0-R_0$
  • $v=0.02$
  • $\beta=0.00005$

Our $\beta$ conditions tell us how many people can be infected depending on the total # of infected. For example, with our $S_0$ and $I_0$, we will know that in the first day, the total number of population that are predicted to be infected is $\beta S_0I_0\approx 5$. In the second day, we are gonna use the new $S_1$ and $I_1$ to calculate the same.

It’s the same case with our $v$ rate. We can look the graph of this prediction

The susceptible population decreases faster, while the recovered grow up and almost erradicated the infected population in 100 days. This is thanks to our $v$ and $\beta$ rate. If we decrease our $v$ rate to $v=0.02$ and our $\beta$ rate to $\beta=0.00002$, we have the following system:

Now, the population takes longer to get infected and longer to get recovered. In this case, probably takes 200 days to control the epidemic.

It’s just a simple look to how two values can control the entire system and gives us different perspectives. In the next examples, i’m going to use the original $v$ and $\beta$ rates.

SIRS model

So, suppose that we wanted to model a desease in which individuals can have a temporary but not permanent immunity after getting infected. We can modify the system of equations to make the recovered individuals susceptible one more time.

\begin{equation} \Large\begin{array}{c} \frac{dS}{dt}=-\beta SI+\gamma R \\\
\frac{dI}{dt}=\beta SI-vI \\\
\frac{dR}{dt}=vI-\gamma R \end{array} \end{equation}

We add a parameter $\gamma$ that represent the rate of conversion from recovered to susceptible per period. This model is now called SIRS model.

Using the Euler method one more time, we had

\begin{equation} \Large\begin{array}{c} S_{i+1}=S_i-\Delta t(\beta S_iI_i-\gamma R_i) \\\
I_{i+1}=I_i+\Delta t(\beta S_iI_i-vI_i) \\\
R_{i+1}=R_i+\Delta t(vI_i-\gamma R_i) \end{array} \end{equation}

We can rewrite the system of equations as:

\begin{equation} \Large\begin{array}{c} S_{i+1}=S_i(1-\Delta t\beta I_i)+\Delta t\gamma R_i \\\
I_{i+1}=I_i(1+\Delta t(\beta S_i-v)) \\\
R_{i+1}=R_i(1-\Delta t\gamma)+\Delta tvI_i \end{array} \end{equation}

@njit()
def sirs(N,dt,beta,v,S0,I0,R0,gamma):
    S,I,R=IC(N,S0,I0,R0)
    for i in range(0,N-1):
        S[i+1]=S[i]*(1-dt*beta*I[i])+dt*gamma*R[i]
        I[i+1]=I[i]*(1+dt*(beta*S[i]-v))
        R[i+1]=R[i]*(1-dt*gamma)+dt*v*I[i]
    return S,I,R

gamma=0.015 #The rate of conversion from recovered to susceptible
S,I,R=sirs(N,dt,beta,v,S0,I0,R0,gamma)

Now, with the conversion we can notice that doesn’t change too much. In this case, the susceptible population doesn’t get to 0 like in the SIR model, now it get’s bigger during the days. Also we can notice how the infected population doesn’t decrease because always have susceptible population to take.

SIRS model with deaths and births.

Now, suppose that we want to check if during all the time of the epidemic, the people can die or new people can born. If we modify the system of equations one more time,we had:

\begin{equation} \Large\begin{array}{c} \frac{dS}{dt}=\Lambda(S+I+R)-\beta SI+\gamma R-\mu S \\\
\frac{dI}{dt}=\beta SI-vI-\mu I \\\
\frac{dR}{dt}=vI-\gamma R-\mu R \end{array} \end{equation}

where $\Lambda$ are the rate of births from the total population that resides in that moment (that is $S,I$ and $R$, remember that now, our system is open and doesn’t conserve $N$). And also, I add $\mu$ that is the rate of individuals who died. In this case, it’s the same for the susceptible, recovered and infected populations.

Using the Euler method one more time, we had

\begin{equation} \Large\begin{array}{c} S_{i+1}=S_i-\Delta t(\beta S_iI_i+\mu S_i-\gamma R_i-\Lambda(S_i+I_i+R_i)) \\\
I_{i+1}=I_i+\Delta t(\beta S_iI_i-vI_i-\mu I_i) \\\
R_{i+1}=R_i+\Delta t(vI_i-\gamma R_i-\mu R_i) \end{array} \end{equation}

We can rewrite the system of equations as:

\begin{equation} \Large\begin{array}{c} S_{i+1}=S_i(1-\Delta t(\beta I_i+\mu-\Lambda))+\Delta t(\gamma+\Lambda) R_i+\Delta t\Lambda I_i \\\
I_{i+1}=I_i(1+\Delta t(\beta S_i-v-\mu)) \\\
R_{i+1}=R_i(1-\Delta t(\gamma+\mu))+\Delta tvI_i \end{array} \end{equation}

@njit()
def sirs_bd(N,dt,beta,v,S0,I0,R0,gamma,lam,mu):
    S,I,R=IC(N,S0,I0,R0)
    for i in range(0,N-1):
        S[i+1]=S[i]*(1-dt*(beta*I[i]+mu-lam))+dt*(gamma+lam)*R[i]+dt*lam*I[i]
        I[i+1]=I[i]*(1+dt*(beta*S[i]-v-mu))
        R[i+1]=R[i]*(1-dt*(gamma+mu))+dt*v*I[i]
    return S,I,R

lam=0.0001 #The rate of new births
mu=0.008   #The rate for death

S,I,R=sirs_bd(N,dt,beta,v,S0,I0,R0,gamma,lam,mu)

And in this case, the susceptible population stays almost how it works in the previous examples, but the infected and recovered population have a significant decrease. Why is that? Well, the susceptible population had a increise in person with the rate of births $\Lambda$. In the other hand, all the three populations had a value tht is substracted, that value is given with the $\mu$ rate.

Now you see? Only the susceptible population had substracted persons but in the same time they added population.
