Simulación del vuelo de un cohete con Python.

January 25, 2021 4 minutos de lectura.
Numerical Methods
Numerical Methods • Python

Este código fue hecho para el curso Practical Numerical Methods with Python. Este código es la solución de la primer tarea.

Las ecuaciones del movimiento de un cohete en vuelo puramente vertical son dadas por:

\begin{equation} \begin{array}{c} \frac{dh}{dt}=v \\\
(m_s+m_p)\frac{dv}{dt}=-(m_s+m_p)g+\dot{m}_pv_e-\frac{1}{2}\rho v|v|AC_D \end{array} \end{equation}

  • $h$ es la altura del cohete.
  • $m_s$=50kg es el peso del proyectil.
  • $g$=9.81 $\frac{m}{s^2}$
  • $\rho$=1.091 $\frac{kg}{m^3}$ es la densidad promedio del aire (asumiendola constante durante el vuelo).
  • $A=\pi r^2$ es la máxima área seccional del cohete, donde $r=0.5$m.
  • $C_D$=0.15 es el coeficiente de arrastre (fricción).
  • $m_{po}=100$kg en el tiempo $t=0$s es el peso inicial de la gasolina del cohete.

donde $\dot{m}_p$ es el coeficiente de gasto de la gasolina, el cual esta dado por la siguiente figura:

La masa restante de la gasolina esta dada como:

\begin{equation} m_p=m_{po}-\int_0^t\dot{m_p}dr \end{equation}

Usando el método de Euler con un paso del tiempo de $\Delta t=0.1$s, crea un código en Python para calcular la altura y velocidad del cohete desde su lanzamiento hasta que choque contra el suelo.

Inicia el código.

Primero, debes importar las librerias a utilizar en el código.

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

Con esto hecho, lo que falta es crear las variables antes mencionadas y darle los valores correctos para este problema.

m=50           #Peso del cohete en kg
g=9.81         #Gravedad
p=1.091        #Densidad del aire
r=0.5          #Radio del área
A=np.pi*(r**2) #Área seccional
ve=325         #Velocidad de escape
Cd=0.15        #Coefficiente de arrastre
dt=0.1         #Paso del tiempo
mp0=100        #Peso inicial de la gasolina en t=0
tf=40          #Tiempo total de simulacion
N=int((tf-0)/dt)+1    #Total de pasos
t=np.linspace(0,tf,N) #Arreglo del tiempo

Para crear $\dot{m}_p$, yo use:

mr=[]
for i in range(0,N):
    if (t[i]<5):
        mr.append(20)
    else:
        mr.append(0)

Ahora, es tiempo de obtener $m_p$ para todo el tiempo simulado. Para eso, solo necesitamos resolver la siguiente integral

\begin{equation} m_p=m_{po}-\int_0^t\dot{m_p}dr \end{equation}

Puede ser resuelta con el método Simpson. Ese método es el siguiente:

\begin{equation} \int_a^bf(x)dx\approx\frac{b-a}{6}\left[f(a)-f\left(\frac{a+b}{2}\right)+f(b)\right] \end{equation}

#Esta función es mr
def f(x):
    if (x<=5):
        return 20
    else:
        return 0

def integral(tf,t0):
    N=int(1e6)             #El numero total de intervalos
    h=abs((tf-t0)/N)       #El incremento entre valores de los intervalos
    P1=(h/3)*(f(t0)+f(tf)) #La parte 1 de la ecuación
    OS=0; ES=0            #Iniciamos la suma de los valores
    #Es tiempo de inicializar el ciclo
    for i in range(1,N-1):
        dx=t0+(i*h) #Aqui se decide cual valor utilizar
        if (i%2==0): #Si es un par, se usa la suma de par
            ES+=2*f(dx)
        else:
            OS+=4*f(dx) #Si es un impar, se usa la suma de impares
    return P1+(h/3)*(OS+ES) #Se regresa el valor final de la integral.

def mp(t):
    return mp0-integral(tf=t,t0=0)

Resolviendo la ODE.

La aproximación a la derivada es:

\begin{equation} \frac{dx}{dt}=\frac{x_{i+1}-x_i}{\Delta t} \end{equation}

Discretizando nuestra ODE, tenemos:

\begin{equation} (m_s+m_p)\frac{v_{i+1}-v_i}{\Delta t}=\dot{m}_pv_e-(m_s+m_p)g-\frac{1}{2}\rho v|v|AC_D \end{equation}

Dejando $v_{i+1}$ en la izquierda y moviendo todo lo demás hacia la derecha, tenemos:

\begin{equation} v_{i+1}=v_i+\Delta t\left(\frac{\dot{m}_pv_e}{m_s+m_p}-\frac{\rho v|v|AC_D}{2(m_s+m_p)}-g\right) \end{equation}

Esta última ecuación es la que programaremos. A su vez, tambien incluiremos la altura del cohete:

\begin{equation} h_{i+1}=h_i+\Delta tv \end{equation}

def euler(mr,ve,m,dt,g,p,A,Cd):
    v=np.zeros(N)
    h=np.zeros(N)
    for i in range(0,N-1):
        F1=(mr[i]*ve)/(m+round(mp(t[i]),2))
        F2=(p*v[i]*abs(v[i])*A*Cd)/(2*(m+round(mp(t[i]),2)))
        v[i+1]=v[i]+dt*(F1-F2-g)
        h[i+1]=h[i]+(dt*v[i])
    return v,h

Con esto resuelto, podemos responder a las preguntas.

Gasolina restante.

Al tiempo $t=$3.2 s, cual es la masa (en kg) de la gasolina restante del cohete?

#Solo debes llamar la función mp
print('The remaint mass of propellant is: ',round(mp(3.2),2))

La respuesta es $m_p=36.0 kg$.

Velocidad máxima

¿Cual es la velocidad máxima del cohete en $\frac{m}{s}$?

Tu puedes saberlo con tan solo llamar la función euler y gráficando los resultados. La gráfica es:

Es fácil ver que la velocidad máxima del cohete ocurre en $t=5$s. Para conocer ese mismo valor en Python, uso la función .max().

print('The maximum velocity is v=',round(v.max(),2),'m/s')

Para calcular el tiempo, tenemos:

#You can check the index when the rocket gets his maximun velocity. To this, 
#you need to convert v to a list with v.tolist() and check the index with .index().
#The resulting index is the same index that you need to introduce in your time array.
print('The time for the maximun velocity is t=',round(t[(v.tolist()).index(v.max())],2),'s')

Y ahora, para conocer cual es la altura del cohete cuando alcanza la velocidad máxima, poder usar lo siguiente

#Use exact the same thing to know at what time occur.
print('The time for the maximun velocity is h=',round(h[(v.tolist()).index(v.max())],2),'m')

Lo que nos dice que la altura es $h=525.52$m.

Altura máxima.

¿Cual es la altura máxima del cohete durante su vuelo?

print('The maximun altitude of the rocket is h=',round(h.max(),2),'m')

Su altura máxima es $h=1334.18$m.

Impact.

¿A que tiempo el cohete va a impactarse contra el suelo?

#Para esto, queremos saber cuando la altura del cohete es <0
print('The time when the rocket impact the ground is t=',t[(next(x for x in range(N) if (h[x]<0)))],'s')

Y con esto, sabemos que ocurre en $t=37.1$ s.
